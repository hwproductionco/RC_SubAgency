Balance Lookup

There are two options to allow your clients' customers to access their account balances or to sign-up for a new loyalty program from your client's own website:

  • Option 1 is easier to implement (It's just a link.)
  • Option 2 will allow the lookup result to be shown inside the client website's page.

Option 1

This is the simplest solution and requires no coding or ability to upload files to their webserver.

However, there is currently no way to customize the display with branding elements.

http://www.referralclix.com/balance.php?a=XXX

Replace the XXX with the client's account ID.

That's all.

If you want to get fancier, you can add the following customizations:

  • Language: Replace AAA in the URL below with the two-letter language code of one of the available languages.
  • Metric Decimal Separator: Add "&sep=," to the URL to make the decimal separator be a comma and the thousands separator be a period.
  • Balance Only: To only return the balance for a campaign, replace CCC with the Campaign ID. Campaign IDs are shown below the "Campaign Name" when you click the "Edit" button to the right of the campaign in the campaign list on the Control Panel screen.
  • To use your own custom CSS definition file, replace DDD with the full URL of where your CSS file is.
http://www.referralclix.com/balance.php?a=XXX&lang=AAA&sep=,&balance_only=CCC&css=DDD

Not every option is required. So for example if you just want to have the balance lookup in Spanish with the comma as the thousands separator, it would look like this:

http://www.referralclix.com/balance.php?a=CLIENT_ACCOUNT_ID&lang=ES&sep=,

Option 2

Gives you much more control, but you need to have certain abilities, or know someone who does, and the client's website needs to fit certain requirements.

Option 2 is pretty simple to implement, and takes only two steps:

Step 1

The first step is to create a new page on your site where the results will be shown. Name it with a .php extention (not .html) - For example: "balance_lookup.php".

Make sure that the content area is at least 550 pixels wide (it likely is.)

Place the following HTML code where content would normally go:

<?php $code = $_POST['code'];
$url = "http://www.referralclix.com/";
$url .= "check_code.php?code=$code";
$url .= "&a=XXX";
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL,"$url");
curl_setopt($ch, CURLOPT_HEADER, false);
curl_exec($ch);
curl_close($ch);
?>

Replace the "XXX" with the client's Account ID.

Requirements

You, your client, or someone else must have the ability to add files to your client's web server (through FTP or some other method.)

The server where your client's site is hosted must has PHP enabled. (The vast majority of Web servers do, and it will be unusual for it not to.)

If neither requirements can be met, please use Option 1, above.

If your client is using a content management tool like Joomla, Mambo, or Drupal, include these steps as separate files inside a Wrapper or iFrame.

Step 2

The second step is to place the following HTML Form on a page of your client's website for customers to enter their card code or unique identifier:

<form action="balance_lookup.php" method="post" name="rewards_form">
Account Number:
<input type="text" name="code" size="24" maxlength="255" border="0">
<input type="submit" name="code_submit_button" value="Look It Up" border="0">
</form>

Replace "balance_lookup.php" with the filename of the file from Step 1 (if you named it differently).

Save and upload the changed files to the client's server. Both files should be on the same directory.

That's all.

If you want to get fancier, you can add the following customization:

  • Language: Replace AAA in the URL below with the two-letter language code of one of the available languages.
  • Metric Decimal Separator: Add "&sep=," to the URL to make the decimal separator be a comma and the thousands separator be a period.
  • Balance Only: To only return the balance for a campaign, replace CCC with the Campaign ID. Campaign IDs are shown below the "Campaign Name" when you click the "Edit" button to the right of the campaign in the campaign list on the Control Panel screen.
  • To use your own custom CSS definition file, replace DDD with the full URL of where your CSS file is.

Template:

<?php $code = $_POST['code'];
$url = "http://www.referralclix.com/check_code.php?code=$code";
$url .= "&a=XXX";
$url .= "&lang=AAA";
$url .= "&sep=,";
$url .= "&balance_only=CCC";
$url .= "&css=DDD";
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, "$url");
curl_setopt($ch, CURLOPT_HEADER, false);
curl_exec($ch);
curl_close($ch);
?>

Remember to replace the "XXX" with the client's Account ID.
