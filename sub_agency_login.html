Cloaked Link

This is the code to create a frames-based HTML page on your site that "cloaks" the true location of the application:

<html>
<head>
<title>Grand Mayan Resort</title>
</head>
<frameset border="0" frameborder="no" framespacing="0" rows="100%">
<frame name="news" noresize src="https://www.referralclix.com">
<noframes>
<body bgcolor="#ffffff">
<p>Your browser is not able to support this application. You need a more modern browser.</p>
</body>
</noframes>
</frameset>
</html>
